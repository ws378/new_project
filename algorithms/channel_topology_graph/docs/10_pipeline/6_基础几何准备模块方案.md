# 基础几何准备模块方案

目标：从原始地图输入中生成后续主线可直接消费的几何世界。  
本方案不负责节点、边、图的建立，只负责输出稳定、可分析、可重复使用的几何基础对象。

## 1. 模块边界

**负责**
- 输入标准化
- 区域裁剪
- 分辨率表达统一
- 自由空间 / 障碍空间生成
- 骨架提取
- 骨架变瘦与短枝修剪
- 输出 `geometry_preparation_result`

**不负责**
- 交汇识别与交汇重建
- 节点与边建立
- 正式建图
- sweep、lane、coverage 规划

## 2. 输入与输出

**输入（对外）**
- 原始地图图像
- 区域约束
- 比例尺与运行参数

**输出（正式对象）**
- `geometry_preparation_result`

建议字段：
- `gray`
- `region_mask`
- `free_mask`
- `obstacle_mask`
- `after_open_mask`
- `skeleton_mask`
- `skeleton_pruned_mask`
- `skeleton_pixels_rc`
- `crop_box_px`
- `resolution_m_per_px`

## 3. 处理流程（高层）

1. 标准化原始输入，统一灰度图、裁剪框与分辨率  
2. 生成区域掩膜、自由空间掩膜与障碍空间掩膜  
3. 对自由空间做基础清理，得到更稳定的可骨架化区域  
4. 从清理后的自由空间提取原始骨架  
5. 对原始骨架做变瘦与短枝修剪  
6. 输出 `geometry_preparation_result`

## 4. 几何准备规则

### 4.1 运行尺度规则
- 本模块所有数组、坐标、分辨率都必须使用运行尺度
- 不允许在算法字段、对象名和伪代码中引入 `*_8x`
- 渲染放大倍率只属于 renderer/export，不属于算法输入输出

### 4.2 几何一致性规则
- `crop_box_px` 必须成为本轮所有局部坐标的共同参考系
- `free_mask` 必须受 `region_mask` 约束
- `obstacle_mask` 与 `free_mask` 不允许大面积重叠

### 4.3 骨架稳定性规则
- skeleton 必须从清理后的自由空间生成
- 变瘦与短枝修剪是必要前置，不是可选优化
- 输出必须稳定，不允许引入随机因素

## 5. 关键几何结果约束

### 5.1 `free_mask`
- 应表达可通行区域
- 不应越过区域约束边界

### 5.2 `skeleton_pruned_mask`
- 应作为后续交汇重建的唯一骨架入口
- 不应保留明显短枝毛刺

### 5.3 `skeleton_pixels_rc`
- 应与 `skeleton_pruned_mask` 一一对应
- 应可直接支持局部 tracing、邻域提取与连通分析

## 6. 失败条件（报错）

- 原始输入为空或尺寸非法
- 裁剪区域为空
- `free_mask` 为空
- skeleton 提取失败
- 短枝修剪后骨架为空
- 分辨率缺失或非法

## 7. 伪代码与数据结构

### 7.1 核心思路

基础几何准备并不是“做几张中间图”，而是建立后续所有算法共享的统一几何世界。  
后续交汇重建不应该再回头自己发明基础掩膜和骨架。

### 7.2 数据结构（概念）

- `InputFrame`: `{gray, crop_box_px, resolution_m_per_px}`
  - `gray` 归一化后的灰度图
  - `crop_box_px` 当前处理区域裁剪框，定义本轮几何世界的统一局部参考系
  - `resolution_m_per_px` 每像素对应的真实米数，后续长度计算统一依赖它
- `SpaceMasks`: `{region_mask, free_mask, obstacle_mask, after_open_mask}`
  - `region_mask` 允许参与算法处理的有效区域
  - `free_mask` 可通行自由空间
  - `obstacle_mask` 不可通行障碍空间
  - `after_open_mask` 对自由空间做基础形态学清理后的结果，用于稳定骨架
- `SkeletonView`: `{skeleton_mask, skeleton_pruned_mask, skeleton_pixels_rc}`
  - `skeleton_mask` 从清理后的自由空间提取出的原始骨架
  - `skeleton_pruned_mask` 做完变瘦与短枝修剪后的最终骨架
  - `skeleton_pixels_rc` 最终骨架像素坐标集合，是后续交汇重建的直接几何入口

**与现有 contracts 的对应关系（逻辑视图映射）**
- `InputFrame`：逻辑上对应 `geometry_preparation_result` 中的输入标准化字段
- `SpaceMasks`：逻辑上对应 `geometry_preparation_result` 中的各类 mask 字段
- `SkeletonView`：逻辑上对应 `geometry_preparation_result` 中的骨架与骨架像素字段

### 7.3 输入输出说明

- 输入（对外）：`raw_map` + `region_constraint` + `config`
  - `raw_map` 原始地图输入
  - `region_constraint` 本轮允许处理的区域限制
  - `config` 运行参数与门限配置
- 输入（模块内部聚合视图）：`InputFrame` + `SpaceMasks`
  - 模块内部先统一输入坐标系、裁剪框与分辨率
  - 再在统一坐标系下建立空间表达和骨架
- 输出：`geometry_preparation_result`
  - 含稳定的灰度图、mask、骨架和分辨率表达
  - 是第 2 步交汇重建的唯一正式几何输入

### 7.4 伪代码

#### 对外入口（接原始输入）

```text
function build_geometry_preparation(raw_map, region_constraint, config):
    # 输入:
    #   raw_map: 原始地图，可为灰度图、彩色图或栅格占据图
    #   region_constraint: 当前任务允许处理的区域约束
    #   config: 本轮运行门限、形态学核、骨架参数
    # 输出:
    #   geometry_preparation_result
    # 约束:
    #   本函数只处理运行尺度几何，不引入任何渲染放大逻辑
    # 失败:
    #   任一关键中间结果为空时必须立即报错，不允许静默返回半成品

    # 1) 统一输入表达。
    # 这一层必须把灰度图、裁剪框和分辨率收敛到同一套运行尺度表达，
    # 否则后续 rc 坐标和长度门限无法稳定对齐。
    frame = normalize_input(raw_map, region_constraint, config)

    # 2) 建立区域、自由空间、障碍空间。
    # 这一层的目标是形成完整的空间表达，而不是直接进入骨架逻辑。
    masks = build_space_masks(frame, config)

    # 3) 对自由空间做基础清理。
    # 这里允许去掉明显毛刺，但不允许破坏主通道拓扑。
    cleaned_masks = clean_free_space(masks, config)

    # 4) 提取原始骨架。
    # 提取结果仍可能包含局部毛刺和短枝，因此这里只负责得到可修剪骨架。
    skeleton = build_skeleton(cleaned_masks, config)

    # 5) 做变瘦与短枝修剪，得到最终可分析骨架。
    # 后续所有交汇重建只能读取这里的最终骨架，不允许回头读 raw skeleton。
    pruned_skeleton = prune_skeleton(skeleton, config)

    # 6) 汇总 geometry_preparation_result。
    # 输出必须包含后续交汇重建所需的最小闭环几何信息。
    return assemble_geometry_preparation_result(frame, masks, cleaned_masks, pruned_skeleton, config)
```

#### 内部聚合（模块内使用）

```text
function normalize_input(raw_map, region_constraint, config):
    # 1) 转灰度。
    # 无论原始输入是什么编码，这一步都统一到单通道灰度表达。
    gray = to_gray(raw_map, config)

    # 2) 计算裁剪框。
    # 裁剪框必须完全落在输入图范围内，且不能产生空区域。
    crop_box_px = derive_crop_box(region_constraint, gray.shape, config)

    # 3) 提取局部灰度图。
    # 后续所有 mask 和 skeleton 都在这个局部区域内构建。
    gray = crop_gray_by_box(gray, crop_box_px)

    # 4) 解析分辨率。
    # 若配置缺失或非法，必须报错，因为后续长度门限和米制换算都依赖它。
    resolution_m_per_px = derive_resolution(config)

    # 5) 返回统一输入帧。
    return InputFrame(gray, crop_box_px, resolution_m_per_px)
```

```text
function build_space_masks(frame, config):
    # 1) 生成有效处理区域。
    # region_mask 定义本轮允许参与算法的像素集合。
    region_mask = build_region_mask(frame, config)

    # 2) 生成障碍掩膜。
    # 该步骤只负责识别不可通行区域，不做骨架相关推理。
    obstacle_mask = build_obstacle_mask(frame, config)

    # 3) 生成自由空间掩膜。
    # 自由空间必须被 region_mask 裁切，并避开 obstacle_mask。
    free_mask = build_free_mask(region_mask, obstacle_mask)

    # 4) 对自由空间做一次开操作。
    # 目的不是改变主通道结构，而是移除小尺度锯齿和孤立毛刺。
    after_open_mask = morphology_open(free_mask, config)

    # 5) 返回空间掩膜集合。
    return SpaceMasks(region_mask, free_mask, obstacle_mask, after_open_mask)
```

```text
function clean_free_space(space_masks, config):
    # 1) 从 after_open_mask 出发继续做小尺度清理。
    # 这里只允许做保守清理，不允许把主通道掐断。
    cleaned_free_mask = remove_small_free_islands(space_masks.after_open_mask, config)

    # 2) 重新约束到 region_mask 内。
    # 防止形态学操作把自由空间膨胀到无效区域。
    cleaned_free_mask = cleaned_free_mask AND space_masks.region_mask

    # 3) 重建障碍掩膜的一致视图。
    # 这里的 obstacle 主要供后续局部几何和 first-hit 使用。
    cleaned_obstacle_mask = invert_under_region(cleaned_free_mask, space_masks.region_mask)

    # 4) 返回清理后的 mask 集合。
    return {
        region_mask = space_masks.region_mask,
        free_mask = cleaned_free_mask,
        obstacle_mask = cleaned_obstacle_mask,
        after_open_mask = cleaned_free_mask,
    }
```

```text
function build_skeleton(space_masks, config):
    # 1) 从清理后的自由空间生成细化骨架。
    # 这一步得到的是 raw skeleton，还不能直接给后续模块使用。
    skeleton_mask = skeletonize(space_masks.after_open_mask)

    # 2) 抽取骨架像素坐标。
    # 这一步输出的是 rc 坐标集合，供后续局部 tracing、邻域查询和连通分析使用。
    skeleton_pixels_rc = extract_pixels_rc(skeleton_mask)

    # 3) 返回 raw skeleton 视图。
    return SkeletonView(skeleton_mask, skeleton_mask, skeleton_pixels_rc)
```

```text
function prune_skeleton(skeleton_view, config):
    # 1) 先做骨架变瘦。
    # 变瘦的目标是减少局部厚骨架引入的虚假邻接关系。
    thinned = thin_skeleton(skeleton_view.skeleton_mask, config)

    # 2) 再做短枝修剪。
    # 这一步必须只移除明显毛刺，不得误删主干通道。
    skeleton_pruned_mask = prune_short_side_branches(thinned, config)

    # 3) 重新提取最终骨架像素坐标。
    skeleton_pixels_rc = extract_pixels_rc(skeleton_pruned_mask)

    # 4) 返回最终骨架视图。
    return SkeletonView(skeleton_view.skeleton_mask, skeleton_pruned_mask, skeleton_pixels_rc)
```

```text
function assemble_geometry_preparation_result(frame, masks, cleaned_masks, pruned_skeleton, config):
    # 1) 汇总所有正式字段。
    result = {
        gray = frame.gray,
        region_mask = masks.region_mask,
        free_mask = cleaned_masks.free_mask,
        obstacle_mask = cleaned_masks.obstacle_mask,
        after_open_mask = cleaned_masks.after_open_mask,
        skeleton_mask = pruned_skeleton.skeleton_mask,
        skeleton_pruned_mask = pruned_skeleton.skeleton_pruned_mask,
        skeleton_pixels_rc = pruned_skeleton.skeleton_pixels_rc,
        crop_box_px = frame.crop_box_px,
        resolution_m_per_px = frame.resolution_m_per_px,
    }

    # 2) 做最小闭环检查。
    # 这里不是可选校验；任何关键字段为空都说明该阶段输出不可供下游使用。
    validate_geometry_preparation_result(result)
    return result
```

```text
function validate_geometry_preparation_result(result):
    if result.gray is empty:
        error("gray is empty")
    if result.free_mask is empty:
        error("free_mask is empty")
    if result.skeleton_pruned_mask is empty:
        error("skeleton_pruned_mask is empty")
    if len(result.skeleton_pixels_rc) == 0:
        error("skeleton_pixels_rc is empty")
    if result.resolution_m_per_px <= 0:
        error("resolution_m_per_px must be positive")
```

```text
function to_gray(raw_map, config):
    # 1) 若输入已经是单通道灰度，直接复用。
    if is_single_channel(raw_map):
        gray = raw_map
    else:
        # 2) 若输入是彩色图，按统一规则转灰度。
        gray = convert_rgb_to_gray(raw_map)

    # 3) 统一到算法约定的数据类型和数值范围。
    gray = normalize_gray_dtype(gray, target_dtype="uint8", target_range=[0, 255])
    return gray
```

```text
function derive_crop_box(region_constraint, image_shape, config):
    # 1) 若有显式区域约束，先从约束生成原始裁剪框。
    if region_constraint is not None:
        crop_box_px = region_constraint_to_box(region_constraint)
    else:
        # 2) 若没有区域约束，则默认取整图范围。
        crop_box_px = full_image_box(image_shape)

    # 3) 将裁剪框裁回图像内部。
    crop_box_px = clip_box_to_image(crop_box_px, image_shape)

    # 4) 裁剪框面积必须大于零。
    if is_empty_box(crop_box_px):
        error("crop_box_px is empty")
    return crop_box_px
```

```text
function build_region_mask(frame, config):
    # 1) 先生成与局部灰度图同尺寸的全 1 mask。
    region_mask = full_true_mask(frame.gray.shape)

    # 2) 若 config 还定义了额外禁区，可在这里扣掉。
    if config.forbidden_regions exists:
        region_mask = subtract_forbidden_regions(region_mask, config.forbidden_regions)

    # 3) 保证 region_mask 非空。
    if count_nonzero(region_mask) == 0:
        error("region_mask is empty")
    return region_mask
```

```text
function build_obstacle_mask(frame, config):
    # 1) 对灰度图做障碍阈值分割。
    obstacle_mask = threshold_obstacles(frame.gray, config.obstacle_threshold)

    # 2) 做一次小尺度闭操作，修补障碍边界缺口。
    obstacle_mask = morphology_close(obstacle_mask, config.obstacle_close_kernel)
    return obstacle_mask
```

```text
function build_free_mask(region_mask, obstacle_mask):
    # 自由空间 = 有效区域 AND 非障碍。
    free_mask = region_mask AND (NOT obstacle_mask)
    if count_nonzero(free_mask) == 0:
        error("free_mask is empty after subtract obstacle")
    return free_mask
```

```text
function morphology_open(free_mask, config):
    # 1) 先腐蚀，去掉小尺度尖角和毛刺。
    eroded = erode_mask(free_mask, config.open_kernel)

    # 2) 再膨胀，恢复主体通道宽度。
    opened = dilate_mask(eroded, config.open_kernel)
    return opened
```

```text
function thin_skeleton(skeleton_mask, config):
    # 反复细化直到骨架宽度趋于 1 像素或达到迭代上限。
    return iterative_thinning(skeleton_mask, config.max_thin_iterations)
```

```text
function prune_short_side_branches(thinned_mask, config):
    # 1) 找到骨架端点和分叉点。
    endpoint_pixels = find_endpoint_pixels(thinned_mask)
    branch_pixels = find_branch_pixels(thinned_mask)

    # 2) 从每个端点向最近分叉追踪一段边链。
    prunable_chains = trace_endpoint_to_branch_chains(thinned_mask, endpoint_pixels, branch_pixels)

    # 3) 删除长度小于门限的短侧枝。
    pruned_mask = remove_short_chains(thinned_mask, prunable_chains, config.short_branch_px)
    return pruned_mask
```
