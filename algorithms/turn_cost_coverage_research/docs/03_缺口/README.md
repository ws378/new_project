# 缺口说明

`03_缺口` 保存当前未闭环问题和历史拒绝路线索引。这里的文档回答“还缺什么、哪些路不要再走”。

## 职责

- 记录当前仍影响进入更大范围 UI 主线的缺口；
- 记录已经验证不适合迁入正式链路的历史路线；
- 为后续立项提供边界，而不是保存长讨论过程。

## 文件组织

| 文件 | 职责 |
| --- | --- |
| `01_当前缺口总表.md` | 当前仍需解决或持续看护的问题。 |
| `02_历史缺口索引.md` | 已拒绝、冻结或只作为参考的历史路线。 |

## 禁止内容

- 不保存每轮讨论全文；
- 不保存执行命令流水；
- 不用“缺口”名义复制旧计划。
