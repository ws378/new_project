# 验证说明

`05_验证` 保存验证体系、验收清单、测试分层和当前基线。这里的文档回答“如何证明当前方案可用”。

## 职责

- 定义 `shelf_aware_turn_cost` 的收敛验收口径；
- 定义测试分层、marker 和 GUI 验收边界；
- 保存当前 `shelf_aware` 与 `shelf_aware_turn_cost` 基线说明。

## 文件组织

| 文件或目录 | 职责 |
| --- | --- |
| `01_shelfAwareTurnCost最终收敛验收清单.md` | 当前验收真值，定义候选增强入口必须满足的收敛验收项。 |
| `02_shelfAwareTurnCost闭环验收看板.md` | 闭环验收口径，不记录当前 WIP 或历史批次。 |
| `03_测试分层与marker规则.md` | 测试分层和 pytest marker 规则。 |
| `04_GUI交互级验收边界与手工验收清单.md` | GUI 交互级验收范围和人工检查项。 |
| `05_只读几何诊断验证记录.md` | 只读几何诊断入口、字段口径、代表性样例索引和限制。 |
| `当前基线/` | 当前可对比的基线说明。 |

## 禁止内容

- 不保存大体积 output；
- 不替代真实测试脚本；
- 不用单张 PNG 作为唯一验收依据。
