# 测试用例说明

本目录用于保存带转角代价覆盖规划研究的测试用例索引。当前自动化验收只围绕官方论文流程和 HiGHS 替代 fractional LP。

目录规划：

```text
paper_reference/
  官方论文仓库 examples/notebooks/experiments 的参考入口或转换说明。

maptools_projects/
  本地 examples/maptools_projects 的项目、area、起点和配置索引。
```

约束：

- 研究阶段可以直接使用第三方代码、数据集和算法实现，不因 license 状态阻塞复现。
- 任何第三方来源都必须记录来源路径、commit/version、license 状态和用途。
- 本地 maptools project 大文件通常不复制到这里，只记录路径和 area 选择；如果研究需要复制，应在 case 说明中标注来源。
- 每个可自动运行 case 应有稳定 case id，并在运行产物中写入 `summary.json`。
- 当前不再维护 synthetic/maptools smoke baseline case；MapTools 接入必须通过官方流程适配验收。
- `summary.json` 输出契约见 `algorithms/turn_cost_coverage_research/docs/04_输出契约与产物校验.md`。
