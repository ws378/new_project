# 带转角代价覆盖规划研究

本目录用于复现《Near-Optimal Coverage Path Planning with Turn Costs》，并评估论文官方流程在当前工程中的可视化与工程化边界。

当前主线只保留：

- 官方 `pcpptc` 流程调用。
- HiGHS 替代官方 `FractionalGridSolver` 的 Gurobi LP 求解。
- 运行产物与 summary 契约。
- 后续 MapTools 既有预处理结果到官方 `PolygonInstance` 的必要适配设计。

不再保留本地 synthetic/maptools smoke baseline、规则图 baseline、spanning-tree baseline 或路径 proxy 指标作为算法入口。

## 目录结构

```text
docs/
  论文复现说明、工程化方案、实验记录。
scripts/
  官方复现、MapTools 官方适配、环境检查入口。
src/
  official_maptools_adapter.py
    既有 MapTools 预处理结果到官方 PolygonInstance 的转换。
  official_replacements/
    HiGHS fractional LP 替代。
  visualization/
    summary 与 artifact manifest 工具。
output/
  运行产物。默认不入 git。
third_party/
  本地第三方源码缓存。默认不入 git。
third_party_notes/
  第三方代码、license、可复现性约束记录。
```

## 正式迁入原则

稳定后的实现若迁入正式 planner，应落位到：

```text
algorithms/coverage_planning/planners/turn_cost_mesh/
```

迁入前必须重新评估：

- 与论文官方实现的差异。
- HiGHS 替代是否允许保留。
- 第三方依赖和 license。
- MapTools 预处理适配边界。
- 性能、失败处理和正式 `CoverageResult` 合同。
