# MapTools Plans 目录说明

本目录保存 `maptools` 子工程的专题计划、PRD 和任务拆解文档。

当前文档类型：

- 可用性改造 PRD
- 可用性改造任务拆解
- 路径/覆盖相关集成方案
- 电子围栏/约束对象升级方案
- 电子围栏对 path 交互的复用方案
- 自由连通区分析与小自由区治理方案

边界约束：

- 这里保存阶段性计划与专题方案，不是长期正式真值文档
- 一旦某项设计已经稳定并成为长期约束，应将结果同步回 `maptools/docs/` 下的正式文档
- 仓库级治理和重构执行文档应留在根层 `docs/refactor_governance/`
