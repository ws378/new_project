# MapTools 文档说明

本目录只服务 `maptools` 地图编辑器子工程。

子工程入口：

- `maptools/README.md`
  - 地图编辑器总体说明、安装、启动与常用操作

正式文档：

- `requirements.md`
  - 地图编辑器需求与范围
- `design.md`
  - 地图编辑器架构与技术设计
- `maptools_usability_quickstart.md`
  - 当前可用的主流程、交互入口与使用说明
- `coverage_planner_modes.md`
  - 编辑器暴露的覆盖路径算法模式说明

计划与专题方案：

- `plans/`
  - maptools 专题 PRD、任务拆解、集成方案
  - 入口文档：`plans/README.md`

约束：

- 本目录只维护 maptools 自身文档
- 算法细节文档应留在对应算法子工程的 `docs/`
- 测试集工程文档应留在 `coverage_dataset/docs/`
